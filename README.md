# mail-csv
* The function of this application will be to send emails automatically from a list in a CSV file<br>

Requirements
------------

* python >= 2.4

Running
------------
fill `my_email = ""` and `to_email = ""` in the code and then write the message where you have it `message = ''' '''`

<code><br>
   python main.py
</code>
